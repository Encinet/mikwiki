## 互通

在服务器里聊天前加上 `#` 可以发送到QQ群

互通消息支持[@在线玩家](#@功能)哦

- 详见 [机器人/互通](../robot/#互通)

## 前缀后缀

> 设置入口在[菜单](../command/#mik)的玩家头(最中间)里

记得用`&`来替换`§`

**支持格式化代码和HEX**

::: tip 有用的链接
格式化代码 https://wiki.biligame.com/mc/%E6%A0%BC%E5%BC%8F%E5%8C%96%E4%BB%A3%E7%A0%81
:::

::: info 例子
HEX使用例子: &#FFAA00 (这个相当于&6)

比如ObcbO的前缀是`&#FF0000R&#00FF00G&#0000FFB`
:::

## @功能

> 示例: @ObcbO你好啊 或 ObcbO你好啊

- 格式: (@)玩家ID

可以和其它消息无空格间隔
